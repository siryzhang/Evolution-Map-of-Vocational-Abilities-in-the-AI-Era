# AI 时代职业与组织能力演化图谱

>
🔗访问链接： [Evolution Map of Vocational Abilities in the AI Era](https://siryzhang.github.io/Evolution-Map-of-Vocational-Abilities-in-the-AI-Era/)
---

## 关于这个项目

这是一个单文件交互式 HTML 可视化工具，用政治经济学的结构视角（生产力 / 生产资料 / 生产关系）横跨四个文明时代，系统性地呈现职业演化的全貌。

每个时代锚定一家代表性企业，完整还原其组织结构，再辅以各领域典型职位，让人一眼看清「一个时代的完整职业图景」，以及从一个时代到下一个时代，职业如何被重塑。

## 可视化结构

### Section 00 · 四个时代职业流变（历史序章）

每个时代的商业团队按**三层结构**完整呈现：

```
▌ 核心业务层   产品 · 技术 · 销售 · 运营执行
◆ 职能支持层   财务 · 人事 · 法务 · 行政 · 采购
— 其他领域     医生 · 教师 · 军人（跨时代锚点）
```

**图谱数据：**
- 57 个职位，横跨 4 个时代
- 42 条演化连线（延续 / 升级迭代 / 转型融合 / 消亡）
- 24 个核心业务职位 · 14 个职能支持职位 · 19 个其他领域职位

**职能演化主轴（可追踪）：**

```
财务线：账房先生 → 会计财务 → 财务分析师 → CFO
法务线：护卫头领 → 法务顾问 → 法务合规   → AI合规法务
人事线：（工业首现）人事专员 → 人力资源  → People Ops
行政线：行政主管 ✕ 消亡（数字化办公终结传统行政）
采购线：库丁仓管 → 采购专员 ✕ 消亡（ERP自动化）
```

### Section 02 · 职位深度拆解

### Section 04 · 组织设计工具箱

## 交互特性

- **轨迹追踪**：悬停任意职位，高亮其完整跨时代演化链（BFS 前向 + 后向遍历），tooltip 显示完整路径串
- **类型过滤**：按延续 / 升级迭代 / 转型融合 / 消亡 / 新兴职位筛选连线
- **入场动画**：连线绘制动画在视口进入时触发
- **跨 Section 联动**：热力图点击职位可跳转至职位拆解详情面板

## 技术实现

- **单文件 HTML**，无外部依赖，无框架，无构建工具

## 设计理念

> 这不是一张「哪些工作会消失」的恐慌清单，而是一张**理解变化结构**的地图。

每一次技术革命都在重塑商业组织的职位构成。真正的问题不是「AI 会不会替代我」，而是「在这一轮重组中，哪些能力会升值，我的演化路径是什么」。

用政治经济学的语言说：生产力的跃迁，必然带来生产资料的重新分配，以及生产关系的结构性重组。职业，是这场重组最具体的载体。
